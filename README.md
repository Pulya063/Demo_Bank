# 💳 Demo Bank

Ласкаво просимо до **Demo Bank**! Це демонстраційний проект банківської системи, що складається з бекенду на **FastAPI** та фронтенду на **Streamlit**. Проект дозволяє керувати банківськими рахунками, транзакціями та переглядати баланс.

## 🚀 Функціонал

- **📋 Управління рахунками**: Створення, оновлення, видалення та пошук рахунків.
- **💸 Транзакції**: Додавання транзакцій з підтримкою різних валют та категорій.
- **📊 Баланс**: Перегляд поточного балансу та історії транзакцій.
- **🔍 Пошук**: Зручний пошук рахунків за різними параметрами.

## 🛠️ Технології

- **Backend**: Python, FastAPI, SQLAlchemy, SQLite
- **Frontend**: Streamlit
- **Tools**: Pytest, Ruff, Black, Isort, Mypy, Makefile

## ⚙️ Встановлення та запуск

### 1. Клонування репозиторію

```bash
git clone <repository-url>
cd Demo_Bank
```

### 2. Встановлення залежностей

Використовуйте `Makefile` для швидкого встановлення:

```bash
make install
```

Або вручну:

```bash
pip install -r requirements-dev.txt
```
> **Примітка**: Переконайтеся, що `streamlit` встановлено (`pip install streamlit`), якщо він відсутній у файлі залежностей.

### 3. Запуск Бекенду

```bash
python -m app.main
```
Сервер запуститься на `http://127.0.0.1:8008`.

### 4. Запуск Фронтенду

У новому терміналі запустіть:

```bash
streamlit run demo_frontend/frontend.py
```
Інтерфейс буде доступний у вашому браузері.

> **Важливо**: Перевірте налаштування `BACKEND_URL` у `demo_frontend/frontend.py`, щоб порт відповідав запущеному бекенду (за замовчуванням у коді може бути `8007`, тоді як сервер стартує на `8008`).

## 🧪 Тестування та перевірка коду

Проект містить набір команд для підтримки якості коду (через `Makefile`):

- **Запуск тестів**: `make test`
- **Перевірка покриття**: `make coverage`
- **Лінтер**: `make lint`
- **Форматування**: `make format`
- **Перевірка типів**: `make typecheck`
